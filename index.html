Warp9

one step beyond existing reactive js libraries

Warp9 is a reactive javascript library for building complex UI. To deal with complexity it provides composability and modularization of views. Also warp9 is built to avoid common pitfall of reactive libraries like memory leaks.

Learn more »

Composability

When we write code our default method to handle complexity is a divide and conquer approach. It can be functional decomposition or aggregation in OOP. Warp9 provides the same concept for UI decomposition/aggregation

View details »

Modularization

When you break UI into pieces the good question is how to reuse and distribute it. With warp9 you can use AMD to distribute not only logic, but a view too.
